## 🚀 Cara Deploy di GitHub Pages

1. Buka repository ini di GitHub
2. Pergi ke **Settings** → **Pages**
3. Di bagian "Source", pilih **Deploy from a branch**
4. Pilih branch **main** dan folder **/ (root)**
5. Klik **Save**
6. Tunggu beberapa menit, website akan live di: `https://tyyyyyyyyyyyyyyyy.github.io/nginepmana.my.id`

## 🌍 Cara Pasang Custom Domain (Nanti)

Setelah membeli domain `nginepmana.my.id`:

1. Di GitHub Pages Settings, masukkan `nginepmana.my.id` di field "Custom domain"
2. Di DNS provider domain kamu, tambahkan:
   - **CNAME record:** `www` → `tyyyyyyyyyyyyyyyy.github.io`
   - **A records** (untuk apex domain):
     - `185.199.108.153`
     - `185.199.109.153`
     - `185.199.110.153`
     - `185.199.111.153`
3. Tunggu propagasi DNS (biasanya 1-24 jam)
4. Centang "Enforce HTTPS" di GitHub Pages settings
